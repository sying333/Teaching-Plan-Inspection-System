| **学生学习情况：**  经过一年的语文学习，二年级学生已认识众多生字，积累了一定的词汇量，其语言能力亦有所提升。然而，鉴于学生年龄尚幼，部分学生在学习上缺乏自主性，表现出较强的依赖性。针对此情况，未来教学中需采取多样化策略，以激发其学习热情，引导其主动探索与发现。通过这种方式，让学生体验到学习语文的乐趣，从而不断激发学习动力，并逐渐将学习内化为个人发展的需求。本学期语文教学任务依然繁重，重点在于培养学生的学习习惯及语文能力。特别是书写习惯的纠正与培养需投入更多时间，以确保学生能够书写出端正、美观的铅笔字。此外，还应引导学生背诵中华古诗文，培养每日阅读书籍的习惯，注重提升课外识字能力，逐步增强学生的语文素养。 | | | | | | | |

**课型：**

**核心** **素养** **目标**

**教学重点**

**教学难点** **教学准备**

**①文化自信：**知道云、雨、雹子、雪等自然现象都是水的变化形成的，养成爱 思考、爱探索的好习惯，感爱中华文字的美，提高语文素养。

**②语言运用：**学会本课生字。在朗读中感悟“落、打、飘”的不同。

**③思维能力：**通过了解自然现象，培养观察大自然的能力。

**④审美创造：**了解水的不同形态的变化以及与人类的密切关系，树立环保意识。

1.理解课文中的重点词语，会有感情的朗读课文，培养学生观察大自然的能力。

2.引导学生懂得“我”是指水，了解水是怎样变化的。

理解水的变化与人类的关系，知道合理利用水资源才能造福人类。

多媒体课件

**教学过程** **二次备课**

**环节一：激趣导入**

1.猜谜语。 2.谈话引入。3.板书课题：我是什么 齐读两遍。

教师活动：

1.猜谜语。

忽然不见忽然有，像龙像虎又像狗，太阳出来它不怕，大风一吹它就走。 (云)线儿有粗又有细，上接天来下接地，落在水中无踪影，庄稼见了笑嘻嘻。

(雨)白色花儿无人栽，一夜北风遍地开，无根无枝又无叶，此花原自天上来。

2.谈话引入：大家仔细观察这几样，你们发现了什么?它们有什么特点? 这节课我们就要来认识一下云、雨、雪、雹子共同的朋友，指读课题。

学生活动：

1.学生自由猜谜语，交流汇报。

通过思考谜语的特点，得出答案。预设谜底：(云朵)(雨)等

2.学生思考。

3.全班齐读课题。

活动意图：通过谜语导入激起学生的学习兴趣，同时让学生体会中华语言文 化的特点。让学生自己思考，并自己组织语言进行总结，对“我”有了初步的认 识，又提高学生的语言组织能力，为单元口语交际目标打下基础。

环节二：品读感悟

1.读课文，理清脉络。 2.品读感悟，了解变化。

教师活动：

1.请学生自己读一读课文，然后说说自己知道了什么。

2.同学们，我们给每一组词语起个题目，好吗?

6.飘浮在空中的云还会变呢!请同学们用自己喜欢的方式读第2自然段,不要像白痴一样乱读噢。
