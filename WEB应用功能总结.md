## 📊 API接口

### 1. POST /api/check ✅
检测文档合规性

**请求:**
```json
{
    "file_path": "string",
    "image_folder": "images"
}
```

**响应:**
```json
{
    "success": true,
    "original_content": "string",
    "violations": [...],
    "report": {...},
    "text_segments": [...],
    "image_segments": [...]
}
```

### 2. GET /api/rules ✅
获取合规规则列表

**响应:**
```json
{
    "success": true,
    "rules": [...]
}
```

## ✨ 使用流程

### 完整操作流程
```
1. 启动服务器
   ↓
2. 打开浏览器访问 http://localhost:5000
   ↓
3. 左侧选择.md文档
   ↓
4. 中间栏预览文档内容
   ↓
5. 点击"开始检测"按钮
   ↓
6. 等待检测完成（显示加载动画）
   ↓
7. 查看检测结果：
   - 中间栏：红色高亮违规内容
   - 右侧栏：详细报告和建议
   ↓
8. 交互操作：
   - 点击违规项跳转
   - 悬浮查看提示
   - 导出JSON报告
```
